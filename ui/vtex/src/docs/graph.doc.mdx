# Graphs

The `Graph` module can render various graph types.

<Graph1A />

```tsx
export const Graph1A = () => {
  const graphA = graph(
    node(1).link(node(2)),
    node(3).link(node(1)),
    node(8).link(node(3)),
    node(2).link(node(3)),
    node(9).link(node(3))
  )
    .width(500)
    .height(500)
    .margins(150, 50);
  return <Graph data={graphA} />;
};
```

## Directed Edges

Directed edges can be rendered by calling the `to` method
on nodes:

<Digraph1 />

```tsx
export const Digraph1 = () => {
  const digraphData = graph(
    node("Sam").to(node("John")),
    node("Lisa").to(node("Sam")),
    node("Kyle").to(node("Soyi")),
    node("Lisa").to(node("Kyle")),
    node("Soyi").to(node("Sam")),
    node("Sam").to(node("Elle"))
  )
    .width(500)
    .height(500)
    .margins(180, 100)
    .nodeStyles({
      radius: 4,
      fill: "tomato",
    })
    .textStyles({
      dx: 15,
      dy: 15,
    });

  return <Graph data={digraphData} />;
};
```

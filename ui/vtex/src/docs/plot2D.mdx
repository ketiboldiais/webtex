# Plot2D

The `Plot2D` module renders functions in $\mathbb{R}^2.$

<Plot2D_Demo1/>

The rendering code:

```tsx
import { axis, f, latex, Plot2D, plot2D } from "@webtex/vtex";

export const PlotFig1 = () => {
  const data = plot2D(
    axis("x").label("x"),
    axis("y").label("y"),
    f("x").equals("2x^2"),
    f("x").equals("cos(x)"),
    latex("f(x) = \\cos(x)").at(-4, -1.5),
    latex("f(x) = 2x^2").at(2, 3)
  )
    .domain(-5, 5)
    .range(-5, 5);

  return <Plot2D data={data} />;
};
```

## Riemann Sums

Riemann sums can be rendered with the `riemann` handler:

<PlotFig2 />

the code:

```tsx
import { axis, f, Plot2D, plot2D, riemann } from "@webtex/vtex";

export const PlotFig2 = () => {
  const data = plot2D(
    axis("x").latex("x"),
    axis("y").latex("y"),
    f("x").equals("x^3").and(riemann(-2, 2))
  )
    .domain(-5, 5)
    .range(-5, 5);

  return <Plot2D data={data}/>;
};
```

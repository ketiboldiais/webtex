# Plot2D

Function plotting is available via the `Plot2D` module:

<Plot2D1 />

```tsx
import { axis, f, Plot2D, plot } from "@webtex/vtex";

export const Plot2D1 = () => {
  const plotData1 = plot(
    axis("x"),
    axis("y"),
    f("x").equals("x^2").stroke("red"),
    f("x").equals("2x - 1").stroke("blue")
  );

  return <Plot2D data={plotData1} />;
};
```

`Plot2D` expects a data type `$PLOT2D` — a mutable `Plot` object. These objects are returned from the function `plot`, which takes $n$ objects of type `Plottable`.

The following are all `Plottable` objects. To instantiate these objects, call their respective handler (listed `Handler`):

| Type          | Description                    | Handler          |
| ------------- | ------------------------------ | ---------------- |
| `$AXIS`       | A mutable `Axis` object.       | `axis`           |
| `$FUNCTION2D` | A mutable `Function2D` object. | `f`              |
| `$ASYMPTOTE`  | A mutable `Asymptote` object.  | `asymptote`      |
| `$POINT2D`    | A mutable `Point2D` object.    | `point`          |
| `$LABEL`      | A mutable `Label` object.      | `label`, `latex` |

## Integrals

Integrals can be rendered by passing an `integral` object to the function’s `and` method:

<Integral1 />

```tsx
export const Integral1 = () => {
  const plotData1 = plot(
    axis("x").ticks(8),
    axis("y").ticks(8),
    f("x")
      .equals("cos(x)/2")
      .stroke("red")
      .and(integral(-3, 3).fill("gold"))
  )
    .domain(-5, 5)
    .range(-1, 1);

  return <Plot2D data={plotData1} />;
};
```

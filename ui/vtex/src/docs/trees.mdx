# Trees

The tree module renders tree diagrams.

<TreeDemo1 />

```typescript
import { leaf, Tree, tree } from "@webtex/vtex";

const TreeDemo1 = () => {
  const treeData = tree("A")
    .nodes(
      leaf("N"),
      leaf("B"),
      tree("J").nodes(leaf("L"), leaf("D"), leaf("F")),
      leaf("K"),
      tree("Z").nodes(leaf("P"), leaf("Q"))
    )
    .curvedLinks();
  return <Tree data={treeData} />;
};
```

## Rendering Curved Links

Tree links can be rendered as straight lines or curved paths.
For the latter, call the `curvedLinks()` method on the root node.
